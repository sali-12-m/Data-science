# Python Foundations Assignment

##  Description
This assignment contains solutions for three Python tasks:
- Reverse a string
- Find the largest number in a list
- CLI-based calculator

### Level 1: Reverse String
- Takes user input
- Reverses the string using slicing

### Level 2: Largest Number
- Accepts a list of numbers
- Finds the largest value using a loop

### Level 3: CLI Calculator
- Supports:
  - Addition
  - Subtraction
  - Multiplication
  - Division
- Includes error handling using try-except
- Prevents division by zero
- Runs in a loop until user exits
